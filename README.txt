Overview
--------
The Bootstrap User Menu module change Drupal My Account menu links to Bootstrap Dropdown.

Installation
------------
Just enable the module.
There are no settings for the module.

Standard module installation applies. See
https://drupal.org/documentation/install/modules-themes/modules-7
for further information.

-- Requirements --

Chaos tool suite (ctools) (https://www.drupal.org/project/ctools)
Bootstrap 3 (https://www.drupal.org/project/bootstrap)


Menus
-----
Make sure you add another "My Account" link to the user menu. We use original link to triger the dropdown.
In the menu item setings set to "Show as expanded".
