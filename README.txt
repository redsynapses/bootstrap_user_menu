Overview
--------
The Bootstrap User Menu module change Drupal My Account menu links to Bootstrap Dropdown.
Add user name and user picture to the menu in place of "My account"

Installation
------------
Just enable the module.
There are no settings for the module.

Standard module installation applies. See
https://drupal.org/documentation/install/modules-themes/modules-7
for further information.

-- Requirements --
Bootstrap 3 based theme.


Menus
-----
Make sure you add another "My Account" link to the user menu since we use original link to triger the dropdown.
In the menu item settings set item to "Show as expanded".
